# Quy tắc Chú thích (Comments)

## Chú thích (Comments)
- Luôn viết chú thích bằng **tiếng Việt**.
- Nội dung: Ngắn gọn, súc tích, dễ hiểu.
- Tránh giải thích dài dòng, tập trung vào giải thích tại sao (tại sao làm thế này) hơn là cái gì (làm cái gì) nếu code đã rõ ràng.

## Ngôn ngữ
- Mã nguồn (biến, hàm, class): Tiếng Anh.
- Tài liệu / Giải thích logic: Tiếng Việt.
